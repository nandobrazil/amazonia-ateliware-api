## Description

Use NODE v18.16.0
AmazonIA Ateliware - API Rest build with NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# First: migrate dev database with sqlite (mysql is used on prod)
$ npx prisma migrate dev

# watch mode
$ npm run start:dev

# prisma
$ npx prisma studio
```

## Test

```bash
# unit tests
$ npm run test

```
