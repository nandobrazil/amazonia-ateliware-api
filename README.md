## Description

Use NODE v18.16.0
AmazonIA Ateliware - API Rest build with NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# prisma
$ npx prisma studio

# migrate dev database with sqlite (mysql is used on prod)
$ npx prisma migrate dev
```

## Test

```bash
# unit tests
$ npm run test

```
